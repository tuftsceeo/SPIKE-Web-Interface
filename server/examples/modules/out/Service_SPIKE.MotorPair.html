(inner) set_motor_rotation(amount, unit)

Sets the ratio of one motor rotation to the distance traveled. If there are no gears used between the motors and the wheels of the Driving Base, then amount is the circumference of one wheel. Calling this method does not affect the Driving Base if it is already currently running. It will only have an effect the next time one of the move or start methods is used.
Parameters:
Name Type Description
amount number
unit string 'cm','in'
